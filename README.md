## Rust安装

### windows

下载地址 `https://www.rust-lang.org/tools/install`

安装文档 `https://rust-lang.github.io/rustup/installation/index.html`

1. 修改环境变量

    - `CARGO_HOME`
    - `RUSTUP_HOME`

2. 确保在运行`rustup-init.exe`前，修改好环境变量
